# 👗 Balaji Dresses Biloli

A modern, fully responsive fashion brand website built with **React** and **Vite**, designed to provide an immersive browsing experience through smooth scrolling, elegant animations, and visually appealing layouts.

The project showcases the Balaji Dresses collection with beautifully crafted sections, animated transitions, and a premium user interface that enhances the overall shopping experience.

---

## ✨ Features

- 🎥 Fullscreen animated landing section
- 🧵 Fashion brand showcase
- 🛍️ Product collection gallery
- 🌟 New Arrivals section
- 📖 About the brand section
- 🎬 Smooth page transitions
- 🎨 Premium UI with Styled Components
- ⚡ High-performance animations using GSAP
- 💫 Framer Motion page animations
- 🖱️ Smooth scrolling using Locomotive Scroll
- 📱 Fully responsive across devices
- 🚀 Fast loading with Vite
- 🎯 Component-based architecture

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/Balaji-Dresses-Biloli.git
```

Navigate into the project

```bash
cd Balaji-Dresses-Biloli
```

Install dependencies

```bash
bun install
```

---

# ▶️ Run Development Server

```bash
bun run dev
```

---

# 📦 Build for Production

```bash
bun run build
```

Preview production build

```bash
bun run preview
```
